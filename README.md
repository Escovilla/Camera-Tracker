
# Camera-Tracker

Camera Tracking using the cheapest and the most easiest way to deploy application

## Prerequisites
Harware Stuff
- Arduino Uno
- L298N (motor drivers)
- Dc Motor

Python
- Mediapipe
- OpenCV

Camera
- Esp32caM
- Or any Camera


## Installation

Install Mediapipe

```bash
  pip install mediapipe
```
Install OpenCv
```
pip install opencv-python
pip install opencv-contrib-python
```
